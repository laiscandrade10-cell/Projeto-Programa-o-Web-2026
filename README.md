Projeto de Programação Web - 2026
Este é o projeto final de Programação Web desenvolvido para a disciplina do professor Samuel Coelho no 2º ano do Ensino Médio.
Sobre o Projeto
O projeto consiste em um site de receitas único, desenvolvido com uma temática visual baseada em art pixel. O objetivo é unir a culinária com uma estética retrô e gamer de forma divertida e intuitiva.
*Tecnologias Utilizadas:
HTML5,
CSS,
JavaScript,
*Temática:
Estilo visual em Pixel Art
